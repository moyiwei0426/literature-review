# ARIS-Lit

A tool-first, model-augmented literature review pipeline.

## Purpose
Build a reliable system for:
- literature retrieval
- PDF parsing
- structured extraction
- claims-evidence matrix generation
- novelty/gap analysis
- review drafting with LaTeX/BibTeX

## Operating rule
Always read these files before continuing implementation:
1. `plans/TASK_BREAKDOWN.md`
2. `progress/STATUS.md`
3. `progress/WORKLOG.md`
4. `docs/ARCHITECTURE.md`

## Canonical project status
- Current source of truth: `progress/STATUS.md`
- History of updates: `progress/WORKLOG.md`
- Next actions queue: `plans/NEXT_ACTIONS.md`
