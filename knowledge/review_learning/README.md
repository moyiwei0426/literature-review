# Review Learning Assets

Purpose: store high-quality review/research writing samples, extracted rhetorical structure, reusable writing patterns, and generation rules for pedestrian-crossing / road-user behavior literature reviews.

## Files
- `corpus_index.json`: collected samples and metadata
- `writing_patterns.md`: distilled prose patterns and section moves
- `section_blueprints.md`: reusable section templates for this domain
- `extraction_schema.json`: schema for per-paper extraction
- `sample_extractions/`: extracted structures from individual papers

## Strategy
1. Learn **prose quality** from high-quality OA papers (including Nature-family when relevant).
2. Learn **review structure** from domain review/survey papers in transportation / ITS / safety literature.
3. Separate:
   - rhetorical moves
   - evidence organization
   - citation behavior
   - gap-writing patterns
   - future-work patterns
4. Feed distilled patterns back into writing prompts and rule-based fallbacks.
